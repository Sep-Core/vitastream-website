"Multi-sensor intelligent diversion system": ensuring preliminary compliance of recycled greywater with standards from all aspects.(pH between 6.0 to 9.0 and turbidity under 5 NTU) Post-treatment via a UV sterilizer and Granular filterenable safe reuse(enabling 99% of particles are filtered out and no microorganisms remaining), with the main controller strictly following national standards to regulate solenoid valves, minimizing misjudgment.
  2. Working Order

Greywater from sinks is collected into the main tank for preliminary disinfection, multi-sensor detection and natural sedimentation. The upper purest water is pumped for direct use; the lower greywater is filtered, flows to the generator to generate electricity. Sensors transmit data to the main board, which controls solenoid valves for diversion; finally, greywater enters corresponding tanks for reuse after power generation.

Software Design

The software system takes cycle detection and state discrimination as the core logic. The main program periodically reads various sensor data and judges the water quality level according to the preset threshold. The judgment result and the liquid level state jointly determine the control strategy of the solenoid valve and pump to realize the automatic diversion, storage or discharge of wastewater.

At the same time, the system uploads real-time data to the mobile phone terminal through the Bluetooth module. Users can intuitively view the pH value, TDS value and the liquid level status of each water tank, and manually intervene in the system operation when necessary, so as to realize the organic combination of automatic control and manual management.
